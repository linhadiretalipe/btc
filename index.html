Guia de Filtros jq

.

Retorna o objeto completo

.campo

Extrai um campo específico

.campo1, .campo2

Múltiplos campos

{ campo1, campo2 }

Novo objeto com campos

.[]

Itera sobre array

[.[] | .campo]

Mapeia campo de array

.[] | select(.amount > 0.5)

Filtra por condição

length

Tamanho de array/objeto
